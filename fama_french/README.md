# Fama-French 5-Factor Data Folder

Place your Fama-French 5-factor Excel or CSV file in this folder.

Once the file is here, we will use it to:
1. Load the Fama-French factor returns (Mkt-RF, SMB, HML, RMW, CMA, and RF).
2. Align the dates of the factor returns with your stock dataset.
3. Train factor models (standard asset pricing regressions) to compare against our machine learning models (FNN, Random Forest, and Ridge).
